# TrustBot — California Living Trust Generator

A web app that generates a complete California estate planning package from a simple intake form.

**Documents generated per client:**
- 📜 Revocable Living Trust (improved, CA Probate Code §§15000 et seq.)
- 🏥 Advance Healthcare Directive (§4701)
- 💼 Durable Financial Power of Attorney (§4401)
- 🔒 HIPAA Authorization
- 📋 Pour-Over Will

---

## Run Locally

```bash
cp .env.example .env   # fill in SMTP credentials
npm install
npm start
# → http://localhost:3000
```

## Share via Tailscale Funnel (instant public URL)

```bash
npm start &
tailscale funnel 3000
# → public HTTPS URL anyone can visit
```

## Deploy to Railway (always-on hosting)

1. Push this repo to GitHub
2. Go to [railway.app](https://railway.app) → **New Project → Deploy from GitHub**
3. Select this repo — Railway auto-detects the Dockerfile
4. Add environment variables in Railway dashboard (copy from `.env.example`):
   - `SMTP_HOST`, `SMTP_PORT`, `SMTP_USER`, `SMTP_PASS`, `EMAIL_FROM`
5. Railway gives you a public URL like `https://trustbot-production.up.railway.app`

> **Free tier:** Railway gives $5/month of free credits — enough for light usage.

---

## Environment Variables

| Variable | Description |
|---|---|
| `PORT` | Server port (default: 3000, auto-set by Railway) |
| `SMTP_HOST` | SMTP server (e.g. `smtp.gmail.com`) |
| `SMTP_PORT` | SMTP port (e.g. `587`) |
| `SMTP_SECURE` | `true` for port 465, `false` for 587 |
| `SMTP_USER` | Your email address |
| `SMTP_PASS` | App password (Gmail) or API key (SendGrid) |
| `EMAIL_FROM` | From name/address shown to recipients |
